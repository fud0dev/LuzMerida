# ⚡ LuzMérida · Mérida XYZ

> **Monitor de precios de la luz por horas en tiempo real.**

**LuzMérida** es un portal de utilidad pública diseñado para ayudar al ciudadano a optimizar su consumo eléctrico. Mediante la visualización de la curva de precios PVPC (Precio Voluntario para el Pequeño Consumidor), el usuario puede identificar fácilmente los tramos más económicos del día.

## 🚀 Características Principales

- **Monitor PVPC Diario**: Visualización de los precios de la luz para las 24 horas del día actual.
- **Alertas de Ahorro**: Identificación automática de las horas con precios mínimos (Ahorro) y máximos (Precaución).
- **Gráfica de Tendencia**: Curva interactiva de precios para visualizar el comportamiento del mercado eléctrico a lo largo del día.
- **Interfaz Premium**: Diseño limpio y funcional integrado en el ecosistema estético de Mérida XYZ.

## 🛠️ Tecnologías Utilizadas

- **Scraper Engine**: Python para la sincronización con fuentes de datos de energía oficiales.
- **Frontend Layer**: HTML5, CSS3 avanzado y Chart.js para la interactividad y visualización de datos.
- **Automation Pipeline**: GitHub Actions para la actualización automática diaria a las 00:05 AM.

## 📊 Fuentes de Datos

La información proviene de fuentes oficiales y abiertas del mercado eléctrico español (ESIOS / Red Eléctrica de España), procesada para ofrecer una lectura simplificada y útil para el hogar.
